### 작업 재개 체크리스트

- `git status --short`로 사용자 변경과 이전 작업 결과를 먼저 확인한다.
- 이 문서의 미완료 후속 과제와 최신 기록을 확인한다.
- 수정 전 실제 사용 중인 구현인지 import 경로를 확인한다.
- 수정 후 최소 `npx tsc --noEmit`, 소스 범위 ESLint, `npm run build`를 실행한다.
- 수행 내용, 변경 파일, 검증 결과, 남은 문제를 이 문서 최상단 최신 기록에 추가한다.

- 작업마다 날짜, 목적, 대상, 수행 내용, 검증 결과를 기록한다.

- 해결되지 않은 문제는 후속 과제로 남기고, 해결 시 해당 기록을 갱신한다.
- 사용자가 이미 작성한 변경은 임의로 수정하거나 작업 성과에 포함하지 않는다.

## 2026-07-16 — 첫 화면 브랜드 리빌 모션 추가

### 목적

페이지 최초 진입 시 브랜드 인상을 만들면서 히어로 화면으로 자연스럽게 연결한다.

### 변경 사항

- 첫 화면에 한 번 표시되는 검은색 브랜드 인트로 오버레이를 추가했다.

- 커튼 퇴장에는 `cubic-bezier(0.76, 0, 0.24, 1)`을 사용했다.
- `prefers-reduced-motion: reduce` 환경에서는 인트로를 즉시 생략한다.
- 인트로는 장식 요소로 처리해 보조 기술에서 제외했다.
